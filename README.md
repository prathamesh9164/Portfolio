## 🌟 Introduction
Welcome to my online portfolio! This application was built to provide a comprehensive view of my professional background. From MERN stack web applications to AI-powered Python scripts like *ACCIDETECT 3*, this platform serves as the central hub for my technical journey.

The UI features a sleek dark/light mode toggle, dynamic interactive backgrounds, and responsive components built primarily with React and Styled-Components.

---

## ✨ Features
- **🌓 Dark/Light Mode:** Seamlessly switch between themes for optimal viewing.
- **👨‍💻 About Me:** A detailed overview of my background, current roles, and passion for technology.
- **💡 Skills Map:** Categorized technical proficiencies including Frontend, Backend, Visualization, and DevOps tools.
- **💼 Experience & Education:** An interactive timeline tracking my academic progress and past professional roles.
- **🚀 Projects Gallery:** Filterable cards showcasing standout projects like the *AI Accident Detection System* and *My Notes*, complete with tags, descriptions, and direct code links.
- **📬 Direct Contact:** An integrated form utilizing EmailJS to send me messages or inquiries directly from the site.

## ⚙️ Installation & Local Setup

To run this portfolio locally on your machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/prathamesh9164/Portfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm start
   ```

5. **View in browser:**
   Open your browser and visit: `http://localhost:3000`
